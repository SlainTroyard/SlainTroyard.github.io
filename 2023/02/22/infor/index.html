Overleaf快速入门

1. Web界面

Overleaf，http://www.overleaf.com/

2. 功能要点

  • 协作编辑Latex，编译Latex
  • 保存在线特定版本，保存离线版本
  • 被编辑链接的共享

3. 一般步骤

  1. 登录Overleaf
  2. 左半部分为编辑区，有半部分为编译区（预览区）。编辑latex，编译（可以自动编译或手动编译）
  3. 在需要时候保存在线版本，在上面的History & Reivisions，版本命名为_，如，180115_1701_Jifeng或180115_1701_Jifeng_before-submission

4. 注意事项

  • 适当的作离线版本保存，Project -> Download As Zip
  • 参考文献的bibtex，建议使用DBLP或Google Scholar。
  • 如有大段文字需要离线编写时，注意告知其他合作者。
  • 通常在Latex文件开头，会配有下面文字，其中Yiqixie链接是用来辅助Latex编写，存储备忘和其它中间内容。

5. 基于意见反馈的协作写作，最佳实践

基于意见反馈的协作，指文字由编辑者负责修改，并由评注者负责提出意见或建议（不参与修改）的修改模式。主要用于文章基本成型之后。

通过下面Latex命令命令，其中\editor或\annotator可以修改为词如\tiger或\tom等，\textcolor{red}中red可以修改为其它颜色。

在Latex中，可以通过如下形式协作。

Original text:

This is One apple.

Annotation:

This is One apple. \annotator{Change One into an.}

After compiling:

This is One apple. Jifeng: Change One into an.（红色）

Editor decision:

This is an apple. \annotator{Change One into an.} \editor{Done. }

After compiling:

This is One apple. Jifeng: Change One into an.（红色）Tom: Done. （绿色）

Annotator feedback:

This is an apple. （由annotator删去自己和editor的评注）

注意，editor不要修改或删除annotator的文字，并需要对annotator的评注逐条回复；而annotator可以修改editor的文字，并删掉editor和自己的评注。

日常工具列表

1. 论文写作

1.1 本地LaTex（如不需要，可跳过）

编辑器，推荐Sublime Text，URL
编译器，推荐Tex Live，URL

1.2 在线LaTex协作（如不需要，可跳过）

推荐Overleaf，URL，只能通过Web端访问

Overleaf快速入门

2. 在线协作

2.1 在线Word与Markdown（如不需要，可跳过）

推荐有道云笔记，note.youdao.com，有Web、手机、PC客户端

2.2 在线Word与Excel（如不需要，可跳过）

推荐一起写，yiqixie.com

3. 项目管理（如不需要，可跳过）

推荐WorkTile, www.worktile.com，有Web、手机、PC客户端

WorkTile快速入门

4. 思维导图（如不需要，可跳过）

推荐XMind，www.xmindchina.net

5. 远程控制（如不需要，可跳过）

推荐TeamViewer, http://www.teamviewer.com/

点击下载

CSTAR-WS服务器使用快速入门

6. 开发管理

推荐Bitbucket与GitHub

6.1 Bitbucket

用于非开源项目，如安全和保密阶段的项目，http://bitbucket.org/

注册Bitbucket账号可能需要访问Google的人机验证工具（出现提示Are you sure you are not a robot）。

6.2 Gitee

用于非开源项目，如安全和保密阶段的项目，http://gitee.com/

6.3 GitHub

用于开源项目，http://github.com/

  • Google Scholar，http://scholar.google.com/

  • Bing，http://bing.com/
  • Baidu Xueshu，http://xueshu.baidu.com/
  • IEEE Xplore，http://ieeexplore.ieee.org/
  • ACM Digital Library，http://dl.acm.org/
  • Springer Link，http://link.springer.com/
  • Elsevier ScienceDirect，http://www.sciencedirect.com/

7.2 DBLP

DBLP，http://dblp.uni-trier.de/db/
